Applications

the fourth chapter in our series on How to Write an Operating System

Your Mission

Provide the mechanism for your OS to handle system calls and write one or two of the simpler calls. Also write a simple application program that utilizes one of these system calls, and have the OS "load" and run it. Don't forget that the boot loader will actually load the file; the OS merely has to interpret the format and jump to the starting address.

Now would be a good time to start work on the system standard library.
